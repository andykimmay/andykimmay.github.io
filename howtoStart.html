Getting Started

What is an API?

API stands for Application Programming Interface. It's a relatively simple concept: an API is how software connects with other software. API's will typically provide a set of routines, protocols, and tools. These allow developers to access the application in a limited but powerful way. APIs allow access to a set of features, including data and functions. Developers can then use these features to create other tools without having to build the features themselves.

Visual Example

Before we get started with the API, let's take a quick visual look at what APIs can do for us.
In addition to the API (which we will use to call data), Pinterest also provides developers certain functions which they call 'Add-Ons'. These 'Add-Ons' provide us a great visual example of what we will do with the API.

JAVASCRIPT TAB: This is the code which allows us to access the Pinterest 'Add-Ons'. This is a generic piece of code that we will need to have on every page where we use the 'Add-Ons'.
HTML TAB: We're going to specifically make use of the 'board widget' which we specify in the HTML. This HTML code is similar to how you would link to a webpage. However, there are additional fields that indicate that this code should embed a 'Board' in the size specified. These additional fields indicate that this should not be treated as a normal link but rather, by pulling on the JavaScript code, that it should be a Pinterest Board.
RESULT TAB: We end up with a Pinterest Board which makes use of the easy and beautiful view that Pinterest has already put together for us.
