Pinterest JavaScript SDK

The previous page discusses how we would normally make API calls. And we can make calls to the Pinterest API using that method. However, Pinterest also provides a software development kit (SDK) that we can use instead, which simplifies a lot of the code. Instead of writing all of that code, we can instead write something like this:

PDK.request('/pins/', 'POST', data, callback);

The code above is a request to the Pinterest API to create a new Pin. Much easier and simpler than the XMLHttpRequest object. Anything that you wanted to do with the API is available through the SDK; more information on what you can do can be found here:
