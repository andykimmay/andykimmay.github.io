Brief Aside on OAuth

Before we dive into the nitty gritty, we should take a quick aside to discuss OAuth. OAuth will allow us to access users' Pinterest accounts and content without telling us their password. How does this work?

  • The Pinterest API allows us to direct our users to Pinterest to ask for their permission.
  • Once the user approves, we will receive an authorization code.
  • We will exchange this code for a token, which is a permanent credential. We can now use this token (by including it on our API calls) to perform actions.
