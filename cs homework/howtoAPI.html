API Calls

So we're all set up to use the Pinterest API. How do we use it? In JavaScript, we make requests through what is known as AJAX (Asynchronous JavaScript and XML). If you want to learn more about AJAX, you can read up on it on the Mozilla Developer Network. In this section, we will take a look at the generic standard format to break down the process. Afterwards, we'll replace the boilerplate language with the specific syntax needed to access the Pinterest API.

We will be working with an object called XMLHttpRequest through which we make calls to the API.

  • First, we start by making the request var request = new XMLHttpRequest();
  • Next, we specify the request type request.open(requestMethod, queryURL + queryString);
  • We are sending two parameters: the request method (GET,POST,PUT,DELETE) and the query. This is how we tell Pinterest what information we want. We need to use the syntax that Pinterest provides in their official documentation to make sure that we get back the data that we want. As mentioned earlier, we'll go over how to do this in the next section.
  • Now we will need our App ID. Pinterst requires a header which contains the App ID with every API request. This way, they know who is making the request. We do this with the following line request.setRequestHeader("Client-ID", apiKey);
  • Then we tell our application what to do once we receive a response. We’ll come back to this later, but this is the format. request.addEventListener(“load”, function() {... } );
  • Finally, we send our request with request.send(null);


This is what it looks like when its all put together: 
var request = new XMLHttpRequest();
request.open(requestMethod, queryURL + queryString);
request.setRequestHeader("App-ID", apiKey); 
request.addEventListener(“load”, function() {...} );
request.send(null);
